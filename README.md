ABOUT THE PROJECT
-----------------
A modular medieval combat system developed in Unreal Engine 5.7.
This project is designed as a clean, readable, and scalable combat foundation
for action and RPG-style games.

The focus is on animation-driven combat, gameplay clarity, and extensible system design.
No AI systems are implemented at this stage.

------------------------------------------------------------

FEATURE OVERVIEW
----------------
• Target / Lock-On System
• Hand-to-hand combat mode
• Sword combat mode
• Directional damage & hit reactions (front / back)
• Niagara VFX for impact and hit feedback
• Synchronized audio with combat states
• Clean and modular Blueprint architecture

DESIGN GOALS
------------
• Clear and readable combat feedback
• Animation-driven combat flow
• Modular and extendable system design
• Suitable as a base for action, RPG, or souls-like combat systems
• Easy to expand with AI, abilities, or multiplayer later

------------------------------------------------------------

TECHNICAL DETAILS
-----------------
Engine        : Unreal Engine 5.7
Core Logic    : Blueprint-based systems
Animation     : Animation Montages & State Machines
VFX           : Niagara Particle System
Audio         : Gameplay-synchronized sound cues
AI            : Not implemented
